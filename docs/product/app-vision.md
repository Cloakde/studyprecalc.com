# App Vision

PrecalcApp is a web practice environment for AP Precalculus students.

## Primary Goals

- Let students answer multiple-choice questions in an exam-like flow.
- Let students answer free-response questions with typed work, structured steps, images, or future drawing support.
- Let students take structured unit practice exams for Units 1-4.
- Let students take AP prep exams that include Units 1-3 only.
- Show answer explanations after submission or during review.
- Show step-by-step solution paths.
- Track attempts, correctness, timing, and review status.
- Surface dashboard-ready progress metadata for retries, recent activity, and class progress summaries.
- Support future video explanations attached to individual questions.

## Initial User Types

- Student practicing AP Precalculus.
- Teacher or tutor reviewing question quality and explanations.
- Content author adding original practice questions.

## Important Boundaries

- The app can use an AP-style workflow, but should not copy protected College Board questions, text, images, scoring rubrics, or branding unless the user has rights to use them.
- AP prep exam mode should exclude Unit 4 by default because Unit 4 is course content but not assessed on the AP Exam. Unit 4 should still have its own course practice exam.
- Question content should be original or explicitly licensed.
- Explanations should teach the underlying precalculus skill, not only reveal the final answer.

## First Useful Version

The first useful version should include:

- Local question bank.
- Multiple-choice question runner.
- Free-response question runner with typed responses.
- Submit and review flow.
- Explanation and step display.
- Basic attempt history.
